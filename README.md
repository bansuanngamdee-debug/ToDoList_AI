# To-Do List AI

โปรแกรม To-Do List สำหรับจัดการงานส่วนตัวด้วย Python (CLI)

## คุณสมบัติ
- เพิ่มงานใหม่
- ดูงานทั้งหมด
- แก้ไขงาน
- ลบงาน
- บันทึก/โหลดงานอัตโนมัติด้วยไฟล์ tasks.json

## ขั้นตอนการติดตั้ง
1. ติดตั้ง Python (แนะนำเวอร์ชัน 3.7 ขึ้นไป)
2. ดาวน์โหลดไฟล์โปรเจกต์นี้ หรือ clone repository:
   ```
   git clone https://github.com/bansuanngamdee-debug/ToDoList_AI.git
   ```
3. เข้าไปในโฟลเดอร์โปรเจกต์

## วิธีการรันโปรแกรม
1. เปิด Terminal หรือ Command Prompt
2. รันคำสั่ง:
   ```
   python main.py
   ```

## ตัวอย่างการใช้งาน
```
===== ToDo List Menu =====
1. เพิ่มงานใหม่
2. ดูงานทั้งหมด
3. แก้ไขงาน
4. ลบงาน
5. ออกจากโปรแกรม
เลือกเมนู (1-5): 1
--- เพิ่มงานใหม่ ---
ชื่อเรื่อง: ส่งการบ้าน
รายละเอียด: ส่งงาน Python
วันครบกำหนด (YYYY-MM-DD): 2026-03-21
เพิ่มงานสำเร็จ!
```

## หมายเหตุ
- ข้อมูลงานจะถูกบันทึกในไฟล์ tasks.json อัตโนมัติเมื่อออกจากโปรแกรม
- หากไม่มีไฟล์ tasks.json โปรแกรมจะเริ่มจากรายการว่าง
